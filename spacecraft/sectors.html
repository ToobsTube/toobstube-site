Where to find it

Every galaxy sector is fixed and identical on every server — only the star layout inside each one is server-specific. Rarity shown is each resource's spawn weight within that sector, not a literal percentage.

Sector data hand-transcribed from spacecraftplanner.com, verified sector-by-sector against the resource counts each sector reports. Pioneer is confirmed in-game rather than site-sourced (it's the fixed tutorial sector, not one of the 22 server-generated regions).
